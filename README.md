# Security Automation Scripts

## Purpose

This repository contains practical PowerShell, Bash, and Python scripts for cybersecurity operations, log review, system checks, and basic reporting automation.

The goal is to demonstrate hands-on security automation readiness for cybersecurity operations, vulnerability management, infrastructure security, and OT/IT support roles.

## Skills Demonstrated

- PowerShell system review
- Bash/Linux log review
- Python log parsing concepts
- Security reporting
- Basic automation workflows
- Windows and Linux administration support

## Tools Used

- PowerShell
- Bash
- Python
- Windows Event Logs
- Linux auth logs
- CSV reporting
